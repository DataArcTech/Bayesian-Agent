# Bayesian-Agent v0.1-v0.4 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the standalone Bayesian-Agent open-source framework from v0.1 through v0.4 without copying GenericAgent code.

**Architecture:** Bayesian-Agent exposes an agent-agnostic core for evidence, posterior belief tracking, skill registry, context building, rewrite policy, and incremental repair. GenericAgent is represented only by an optional adapter boundary and documentation, not vendored code.

**Tech Stack:** Python 3.10+, standard library, pytest, JSON schemas, CLI via `argparse`.
